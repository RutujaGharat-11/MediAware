MediAware
=========
About
-----
MediAware is a small static web project that helps browse and inspect a medicines dataset. It demonstrates practical front-end skills using plain HTML, CSS, and JavaScript.

Tech stack
----------
- HTML
- CSS
- JavaScript

What you'll see
---------------
- A simple responsive UI in index.html.
- Client-side logic in script.js that reads medicine.json.

Quick start
-----------
Open the project in a browser (double-click index.html) or run a lightweight server:

```bash
python -m http.server 8000
# then open http://localhost:8000
```

Why this project matters
-------------------------------------------
- Shows competency with front-end fundamentals and working with JSON data.
- Clean, minimal codebase easy to review during interviews.
